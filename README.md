# vue-drag

Drag and drop to change the position of the DOM.

## Installation

### With npm or yarn

```
yarn add vue-drag

npm insatll vue-drag
```

## Usage

### Basic

```html
<template>
  <VueDrag>
    Drag me.
  </VueDrag>
</template>

<script>
import VueDrag from 'vue-drag'

export default {
  components: { VueDrag }
}
</script>
```
